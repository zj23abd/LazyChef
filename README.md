LazyChef - A minimalist web-based recipe platform for time-constrained users.

Built with HTML5, CSS3, and vanilla JavaScript. No build steps or dependencies required.

How to Run LazyChef

Download or unzip the folder

Open index.html in any web browser (Chrome or Firefox recommended)

No installation or server required

On first load, an internet connection is recommended to fetch additional recipes from the Spoonacular API. These will be cached locally for offline use afterwards

If no internet is available, 30 built-in recipes will load automatically and all features remain fully functional

Note: Additional recipes are fetched via the Spoonacular API on first load. If the daily API limit has been reached, 30 built-in recipes will load automatically. All features remain fully functional with the built-in dataset.

Files: index.html, style.css, script.js
